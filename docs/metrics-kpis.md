# Learning Metrics

## Pipeline Quality
- Ingestion success rate on sample docs.
- Chunk coverage (did we capture key sections?).
- Retrieval recall on a small QA set.

## Answer Quality
- Citation accuracy (answers supported by sources).
- Faithfulness (no hallucinated claims).
- "Not enough info" rate when sources are weak.

## Performance (Local)
- Query latency on laptop hardware.
- Ingestion time per document.
